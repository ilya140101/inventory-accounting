		Программа товароучета, версия 1.0
Инструкция по эксплуатации:
1. База данных.
	a) Загрузить новую базу данных => Загрузка новой базы данных, с удалением старой.
	б) Добавить товары в текущую базу данных => Добавление новый товаров в текущую базу данных.
2. Документы.
	a) Общий журнал документов => Все отчеты, которые были созданы.
3. Номенклатура. => Список все товаров с остатками в текущей базе.
4. Служебные.
	a) Калькулятор.
	б) Календарь.

Во всех таблицах реализован поиск по всем колонкам.
Для выбора товара в подборе нужно 2 раза кликнуть мышью по строчке. Чтобы удалить товар из выбранных, нужно нажать клавишу "Delete" или "Backspace".

Типы отчетов:
a) Отчет продаж. В данном отчете выбираются товары из базы, с возможностью указания количества, скидки и суммы. Итоговая сумма считается по розничным ценам.
б) Поступление. В данном отчете можно указать прибывшие товары, с возможностью указания количества и с изменением цены и процента начисления при продаже. Итоговая сумма считается по закупочным ценам.
в) Списание. Списание товара, с указанием количества. Итоговая сумма считается по закупочным ценам.

Меню подбор:
Верхняя таблица - таблица с товарами из базы данных.
Нижняя таблица - выбранные товары.

В меню подбор в отчете "Поступление" и в номенклатуре можно добавить новую позицию в базу данных. Код генерируется случайным образом, без повторений.
